## 1) Descripción del proyecto
**Esta tienda online** es una aplicación web que permite a los usuarios buscar, navehar y comprar los móviles.
## 2) Motivo de la elección
Yo he elegido este proyecto porque todos sabemos que las herramientas electronicas están creciendo y son una parte de nuestra vida especialmente los móviles donde ahora raras veces encontramos a una persona que no usa el móvil.
## 3) Funcionalidades de la tienda
- Catálogo interactivo con filtros por marca, precio y características.  
- Carrito de compras y sistema de pago seguro.  
- Panel de administración para gestionar productos y pedidos.
## 4) Tecnologías
### Frontend
- Lenguajes: HTML, CSS y JavaScript
- Framework: [Vue.js](https://vuejs.org/)
### Backend
- Lenguaje: PHP
- Framework: [Laravel](https://laravel.com/)
### Base de datos
- MySQL para los productos y los usuarios

## 7) Fases de desarrollo
1. Diseño de la interfaz en Figma.  
2. Implementación del catálogo en HTML/CSS.  
3. Integración del carrito y sistema de pagos.  
4. Desarrollo del backend y conexión con la base de datos.  
5. Pruebas de accesibilidad y despliegue en GitHub Pages.
